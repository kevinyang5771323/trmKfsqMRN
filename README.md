## 技术栈

- **后端框架**：Django
- **数据库**：MySQL
- **前端图表**：Echarts
- **开发语言**：Python

- **数据爬取**：高效爬取2024招聘网站数据
- **数据分析**：深度解析招聘信息，包括学历、工作经验、技能要求及薪资待遇等
- **数据可视化**：构建交互式界面，提供数据的可视化展示

## 系统角色

系统主要面向以下用户角色：

- **普通用户**：查看招聘推荐信息，进行查询与过滤操作
